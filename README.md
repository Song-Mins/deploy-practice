## 📚 Spring Boot 프로젝트를 ec2 서버에 배포하는 다양한 방법 실습
협업 프로젝트를 진행하고 배포해보면서 다른 배포 방법들에 대해 궁금중이 생겼고 다양한 배포 방법들을 직접 경험해보고 정리하기 위해서 이 프로젝트를 하게 되었다.
</br>

## ✏️ 배포 방법
먼저 스프링 부트 프로젝트를 ec2 서버에 배포하는 방법은 크게 수동 배포와 자동 배포로 나뉜다.

수동 배포에는 매우 다양한 방법이 있는데 이 프로젝트 에서는 일단 아래의 3가지 방법을 직접 해볼 것이다.
1. jar 파일을 ec2 서버에 전송하여 실행
2. ec2 서버에서 깃 클론으로 jar 파일을 생성하고 실행
3. 도커 이미지를 ec2 서버에 전송하여 실행
   
그리고 자동 배포는 CI/CD 도구를 활용해 위의 수동 배포 작업을 자동화하는 것이다. 
이 프로젝트에서는 GitHub Actions 를 사용하여 배포 프로세스를 자동화 할것이다.
